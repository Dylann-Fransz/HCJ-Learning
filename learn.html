Chapter One

<Basic HTML>

What is HTML?

HTML is used for the layout and structure of a website. It does not require a lot of skill or knowledge to use. Meaning it is easy to learn for beginners.

How to use HTML?

Open a code editor like 'Replit.com' and proceed to make a new project. The first page of a website is usually called 'index.html'.

Tags

Every object (text, images, etc.) starts with < followed by the tag name like h1 and ends with >

The final result will look like this: <h1>

It also needs to close. The closing looks the same, it just needs a '/' before the tag name. For example: <h1></h1>

In between you can put things, in this case text as it is a title tag.

<h1>This is a title</h1>

There are many more tags with many more ussages. This is a basic tutorial so I won't get into that. You can check them out at the 'Learn more' link.

Chapter Two

<Basic CSS>

What is CSS?

CSS is used for styling the website. It is used to make the website look better and more attractive. The color of text, the background color, the font, the position of the text etc. They are all CSS properties.

How to use CSS?

HTML is required for CSS to work. If there is no HTML then CSS can not be applied. It's like having butter but no bread. The butter requires the bread. But the bread alone is very bland. So you need butter to add taste. In this case CSS.

So where do you write CSS? The easy option is to add the CSS within the same file the HTML is. It will make things messy and overwhelming, it also isn't great when having multiple pages. So you can add CSS to a separate file and link it to the HTML.

This is a more complex step that can be seen in the Learn more link.

Classes and ID's

In HTML you can use the class attribute to give a tag a class. The class can be used to identify a certain tag. You can add a class attribute with: class=""

Using the tag we used in learning HTML it can look like the following:

<h1 class="">This is a title</h1>

In this case, the <h1> tag is used for a title. Meaning the class should be

<h1 class="title">This is a title</h1>

Why do we need classes? Without them it will be hard to style specific things. Let's say you have two titles, but one needs to be red and the other needs to be green. CSS cannot tell them apart without adding a class.

Same for id's, it is used to specify a specific tag. ID's are written differently. There isn't much of a difference. class="" simply gets replaced with id="".

In your actual styles or css file you select the classes and id's with a special character. For classes you use a '.' and for ID's you use a '#'. Below is an example of what a class and an ID can look like in your CSS.

Chapter Three

<Basic Javascript>

What is Javascript?

Javascript brings functionality to the website. Everytime you click a button a function can be attached to it. For example: a button changes it's color when you have pressed it. This is something Javascript does.

To showcase it, try clicking the button below to change the background of the button.

How to use Javascript?

Javascript can be added just like CSS, for CSS you use style but for Javascript you use script. Javascript can also be added to the HTML file itself, but is easier when adding it to an extern file.

Variables

Javascript uses Variables, a variable is almost like a term with a meaning. In this case the meaning of the term is a Value and the term is the Variable.

There are many variable types, to keep it simple i'll explain two of them, you'll most likely only use these two when starting.

The first variable is called const, this is short for constant. The name says it all, it's a constant value, it cannot be changed later on.

The second variable is called var, this is short for variable, meaning you can change it later on.

Here is once again an example of how const and var are used.

Script:

const Constant_Name = "This is a const value";

 var Variable_Name = "This is a var value";

 console.log(Constant_Name, Variable_Name);

Output

This is a const value

 This is a var value

Functions & Connections

Functions can be attached to connections, what does that mean though?
Let's say you have a button that needs to be clicked, the Connection is clicking the button. What happens after clicking the button is the Function.

Functions can also be called without a connection. In this case you only make a function if you're going to repeatedly use that bit of code inside the function. Otherwise you have to rewrite that code everytime you want to use it. A function on it's own can also be used to clean up your code and make it more readable.

The code inside this function spells a word in reverse, you would want to do this multiple times that's why it's inside of a function.
